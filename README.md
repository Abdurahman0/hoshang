# Solar CRM Frontend

Frontend for Solar CRM (admin/developer panel only).

## Source of Truth

Backend/API mapping is defined in:

- `Solar CRM Frontend Pages.txt`

Template documentation from the original Solar clone has been removed.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run typecheck
npm run build
```

## Scope

- Includes: CRM admin/developer pages
- Excludes: Telegram WebApp (separate project)
